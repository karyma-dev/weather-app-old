# Weather App

Weather application created with [openweathermap api](https://openweathermap.org) to practice react hooks and context api.

## Getting Started

### Installing

```
  git clone https://github.com/karyma101/weather-app.git
  cd weather-app
  npm install
```

### Running Dev Server

```
  npm run dev
```
